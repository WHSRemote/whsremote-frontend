# WHSRemote
Helping Wayland High students navigate remote learning.

YouTube Playlist: https://www.youtube.com/playlist?list=PL5Y2tGfke7b720jp0GcGiOTQhT_SCItpA

## How To Run
1. `git clone`
2. `npm install`
3. `npm run build`
4. `HTTPS=true npm start`
